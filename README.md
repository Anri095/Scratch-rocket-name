# Scratch-rocket-name
My first scratch game
# Description
In this game, the player controls a rocket and tries to avoid incoming asteroids.
Each time an asteroid is successfully avoided, the player earns 1 point.
The goal is to survive as long as possible and get the highest score.
# Features
Rocket movement controls
Falling/moving asteroids
Score system
Game over on collision

Game logic
